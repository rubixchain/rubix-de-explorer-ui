# Rubix Explorer

A comprehensive blockchain explorer for the Rubix ecosystem, providing real-time insights into transactions, tokens, DIDs, and network metrics with an intuitive user interface.

## Project Overview

The Rubix Explorer is a modern blockchain explorer designed to provide comprehensive insights into the Rubix ecosystem. It serves as a central hub for users to explore transactions, tokens, DIDs (Decentralized Identifiers), and network metrics across different chains within the Rubix universe.

### Key Features
- **Multi-Chain Support**: Navigate between different token chains in the Rubix ecosystem
- **Real-time Data**: Live transaction feeds, token balances, and network metrics
- **Advanced Search**: Search by DID, token ID, transaction ID, or block height
- **Interactive Visualizations**: Dynamic network visualization and transaction graphs
- **Comprehensive Analytics**: On-chain metrics, validator information, and pledge distribution

## User Flow

### Primary User Journeys

1. **Landing Page Exploration**
   - User arrives at homepage
   - Views key metrics (total transactions, value settled, DIDs, 24h activity)
   - Explores latest transactions graph with time filters
   - Switches between transaction/block/holder views

2. **DID Search & Analysis**
   - User searches for specific DID
   - Views token balances and total value
   - Analyzes transaction history
   - Explores pledged tokens and validator participation

3. **Token Investigation**
   - User searches for token ID (RBT/FT/NFT/SC)
   - Reviews token details (owner, state, block height)
   - Examines transaction history
   - Tracks token chain evolution

4. **Transaction Deep Dive**
   - User searches for transaction ID
   - Reviews transaction details (sender, receiver, type)
   - Analyzes validator participation
   - Tracks transaction timing and completion

5. **Network Overview**
   - User explores validator network
   - Reviews pledge distribution
   - Analyzes network health metrics
   - Switches between different chains
